Documentação da API

Rota /noticias/id/:id
Rota para listar todos os usuários cadastrados

Essa rota lista todos os usuários do banco de dados com os atributos id_usuário, titulo, conteudo, autor e image.

Exemplo de utilização da rota /noticias/id/:idMetodo GET

"id": 1, "titulo": "eu nãoquero nem saber", "conteudo": "eu sou jogador de bask", "autor": "mirandaaaaaaaaaaaaaaaaaaaa", "image": "https://images.pexels.com/photos/6894428/pexels-photo-6894428.jpeg?auto=compress&cs=tinysrgb&dpr=1&w=500"}]
